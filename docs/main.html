    General

    Use top left layer button to change between map layer options

    Edit legend labels: double click label text

    Save and Load

    New admin map timeline wipes clean current map and replace basemap with admin-1 level world map

    New nation map timeline wipes clean current map and replace basemap with nation level world map

    New map from file wipes clean current map and prompts user to upload a custom geojson basemap; the save of such map will include the custom geojson as well, hence the save will be much larger than the admin/nation map

    Colouring

    Use single fill for filling single province, use lasso fill to fill or delete multiple provinces

    Currently chosen colour will have a solid black border and be protruding

    Colour of currently hovered over region will have a dashed black border and be protruding

    Colours in the toolbar will have a double border if they are on the map already

    Choose a colour: either from colour bar above map or from legend

    Colour region: choose a colour and a date, then simply click the region

    Timeline

    Add date/event entry: Shift + Enter

    Delete date/event entry: Shift + Backspace
